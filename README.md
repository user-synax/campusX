## What is CampusX?

Every Indian college student lives across 5 different platforms вҖ” WhatsApp for updates, Instagram for photos, LinkedIn for fake achievements, Telegram for notes, and Google Forms for events. None of them are built *for students*.

**CampusX** is a student-only social network where you need a verified college email to join. Real identity. Real campus community. No recruiters, no ads, no noise вҖ” just your college, your people, and your content.

> Currently targeting **IGNOU** and **DTU** for soft launch. Built by a solo developer in Delhi.

## Core Features

### рҹҺ“ Student Verification
```
College Email вҶ’ OTP Sent вҶ’ Verified Badge вң“
```
Only real students get in. No fake accounts, no outsiders. Verified badge shows on every post and profile.

---

### рҹ“° Smart Feed Algorithm
Posts are ranked by a weighted score вҖ” not just chronological order:

```
Feed Score = (Likes Г— 1.5) + (Comments Г— 2) + (Views Г— 0.5)
           + Recency Boost + Connection Boost
```

Students see relevant content from their college first, trending content second.

---

### рҹ’¬ Real-time Features
- **Live notifications** вҖ” instant alerts for likes, comments, follows
- **Notification sound** вҖ” toggle on/off
- **Trending sidebar** вҖ” top posts updating live
- **View counts** вҖ” see how many students viewed your post

---

### рҹҸҶ Campus Coins (Virtual Currency)
```
Post content        вҶ’ Earn coins
Get likes           вҶ’ Earn coins  
Daily login         вҶ’ Earn coins
Redeem coins        вҶ’ Unlock features
```
Gamified engagement that rewards quality contribution.

---

### рҹӣЎпёҸ Admin Moderation Panel
Full admin dashboard to manage content, users, and reports вҖ” keeping the campus feed clean and safe.

## Getting Started

### Prerequisites
- Node.js 18+
- MongoDB Atlas account (free tier)
- Pusher account (free tier)
- UploadThing account (free tier)

### Setup

```bash
# Clone the repo
git clone https://github.com/yourusername/campusx.git
cd campusx

# Install dependencies
npm install

# Create environment file
cp .env.example .env.local
```

### Environment Variables

```env
# MongoDB
MONGODB_URI=mongodb+srv://...

# JWT
JWT_SECRET=your_super_secret_key

# Pusher
PUSHER_APP_ID=
PUSHER_KEY=
PUSHER_SECRET=
PUSHER_CLUSTER=
NEXT_PUBLIC_PUSHER_KEY=
NEXT_PUBLIC_PUSHER_CLUSTER=

# UploadThing
UPLOADTHING_SECRET=
UPLOADTHING_APP_ID=

# App
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

```bash
# Start development server
npm run dev
```

Open `http://localhost:3000`

## Security

CampusX is built with security as a first principle:

- **HTTP-only cookies** вҖ” JWT tokens never accessible via JavaScript
- **College email verification** вҖ” OTP-based, blocks fake signups
- **Rate limiting** on auth routes вҖ” prevents brute force
- **Input sanitization** on all API routes
- **Admin-only routes** protected by role middleware
- **No sensitive data in client** вҖ” all secrets server-side only

---

## The Problem We're Solving

Indian college students have no dedicated digital home:

- **WhatsApp groups** вҖ” chaotic, no content discovery, admin-controlled
- **Instagram** вҖ” algorithm hides college content, not student-specific
- **LinkedIn** вҖ” professional pressure, no casual peer connection
- **Telegram** вҖ” anonymous, unsafe, no verified identity

CampusX gives students a **verified, safe, college-first** social space.

**Target market:** 40 million+ college students in India.  
**Initial focus:** IGNOU (4M students) and DTU, Delhi.
