# OG Images

This folder contains Open Graph images for social media sharing.

## Required Files

### default.png (1200x630px)
- Dark background (#0f0f0f)
- CampusX logo/text (large, center)
- Tagline: "India ka Student Social Network"
- Subtle gradient or pattern

## Creating the OG Image

1. Open `default.svg` in Figma or Canva
2. Export as PNG at 1200x630px
3. Save as `public/og/default.png`
4. Verify at: https://opengraph.xyz/

## Tools

- **Figma**: Open SVG, export as PNG
- **Canva**: Import SVG, customize, export
- **CLI**: `convert default.svg -resize 1200x630 default.png` (ImageMagick)
